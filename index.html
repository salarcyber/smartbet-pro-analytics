üìö How to Read This Analysis

Edge: The Expected Value (EV) of a bet. Positive edge = value bet opportunity.

  • Edge > 10%: Excellent value (rare, bet aggressively)
  • Edge 5-10%: Good value (standard value bet)
  • Edge 0-5%: Fair bet (borderline, bet small or skip)
  • Edge < 0%: No value (avoid, negative EV)

Kelly Criterion: Optimal bet sizing as % of bankroll. We use Quarter Kelly (0.25x) for conservative bankroll management.

xG (Expected Goals): Statistical measure of shot quality based on historical conversion rates. Higher xG = more dangerous attack.

Elo Rating: Team strength rating adjusted for current form (last 5 games), league position, and points accumulated.

Poisson Distribution: Mathematical model that calculates goal probabilities. Used by professional bettors to find value.
